# How to use

If there is an existing sync settings, use that. 

1. On the left side, open the file path of the local items. 
2. On the right side, open the backup location
3. Click F8 to open synchronisation settings
4. Select the variant. Variant difference noted here: https://freefilesync.org/tutorials.php
